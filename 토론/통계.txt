- 추가
1. 방 개설시 연관단어 추가하여 정보 분석에 활용 (메타태그 활용)
2. 동일 아이디로 신고 접수 3회 이상이면 강퇴 권한 적용
: 신고된 유저는 신고되었다는 노티를 받음
3. 자료를 파일 저장 (엑셀)
4. + 프린트 출력

- 서비스
1. 관련 단어를 언론사 기사 링크와 연결 (오픈)
: 클릭 시 수입 발생
2. 트렌드 분석 정보 제공 서비스 (클로즈)
: 대쉬보드 현황 제공
성별, 시간별, 사건별, 주제별, 디바이스별, 주제별, + 지역별, 

- 통계 종류
1. 마이페이지 (개인)
	- 토론방 (히트 / 결과)
	: 각 카테고리별 히트 주제 선정 (조회수 / 투표율 / 의견 개수)
	* : 결과 (투표 / + 가장 많이 나온 의견 / 기간 /  )
	- 활동 지수 (선호채널)  
	: 유저가 활동한 지수
	: 선호채널에 대한 진보 / 보수 성향 분석 통계
	+ 다른 유저와의 관계 (입장)
	: 진보 / 보수 성향 표시
	- 회원 / 비회원
		- 방문 페이지
		- 접속시간
		- 검색어
		- 방문 토론방 주제
	- 블랙리스트 
	: 내가 개설한 방에서 신고된 유저를 제어

2. 관리자 (웹로그 분석)
	+ 유입된 경로
	+ 접속 지역 (로컬)
	+ 트래픽 (접속시간대 / 방문통계 / 페이지)
		- 접속 시간대 (최초 접속 시간 / 머문 시간 / 종료 시간 )
	
	- 히트 통계 (히트 / 카테고리 / 개설방)
	- 회원현황 (시간 / 일 / 월)
		- 가입자 (신규 / 누적 / 탈퇴) !
		- 전체 현재 접속자 수 / 누적 !
		- 각 페이지별 현재 접속자 수 / 누적 !
		- 디바이스 별 (모바일 / PC ) !
		- 접속 브라우저 및 OS	!
		- 유형분석 (보수 / 진보)	
			- 개인		x
			- 카테고리		!

	- 서비스 이용 통계 (개설수, 진행중, 종료, 종료임박, 불량토론)
		- 일반 토론방 
		- 1:1 토론방
		- 통계정보 판매 실적

	- 기사 / 단어 유형분석 (전체 / 카테고리 / 성별 / 시간별)
	: 많이 나온 키워드 트렌드 분석하고 해당 키워드 클릭시 뉴스 기사 아웃링크

3. 토론방
	- 참여자 수
	- 조회수
	- 성별
	- 결과
	- 참여도 (의견 / 투표)
	- 비슷한 의견
	- 주제
	- 단어
	- 기사
	: 주제와 관련된 기사를 통계로 보여줌 (: 순위)
	- 날짜별 (: 주, 월, 일)
	- 종료
	: 관리자가 현황 파악
	- 검색어
